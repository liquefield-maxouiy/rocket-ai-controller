Overview

Что такое NOVA?

NOVA — это полностью автономная система управления ракетой на базе нейросети, обученной методом подражания (behavioral cloning). Система работает на микроконтроллере ESP32 и управляет 4 рулями в X-конфигурации для активной стабилизации.

🧠

Нейросетевой контроллер

Глубокая нейросеть архитектуры 11 → 200 → 150 → 80 → 3 с LayerNorm и ReLU/Tanh активациями. Обучена на 30,000+ симулированных полётах с PD-политикой и exploration.

🎯

6-DOF Симулятор

Физически точный симулятор с RK4-интегрированием, реалистичной аэродинамикой, переменной массой/инерцией и 10 видами неидеальностей для робастности.

⚡

Real-time на ESP32

Работает на ESP32 с частотой 50 Гц. Общий цикл: чтение MPU6050 + BMP280 → фильтр Маджвика → нейросеть → сервы. Расход RAM ~320 КБ.

Конечный автомат полёта

Семь состояний с защищёнными переходами, детектом апогея, отсечки двигателя и посадки

INIT → GROUND → ARMED → ASCENT → COAST → DESCENT → RECOVERY

# FastRocketSim — 6-DOF симулятор с неидеальностями

class FastRocketSim:
    def reset(self) → np.ndarray:
        # Масса, аэродинамика, ветер, дрейф — всё с разбросом
        mass_factor = np.random.uniform(0.8, 1.2)
        self.mass_dry = self.base_mass_dry * mass_factor
        self.wind_process = WindProcess(mean_wind, theta, sigma)
        self.gyro_bias = np.deg2rad(np.random.uniform(-1.0, 1.0, 3))

    def step(self, action):
        # RK4 + ветер + шум датчиков + сбои I²C
        self.wind = self.wind_process.step(dt)
        acc_noisy = acc_body + self.acc_bias + np.random.randn(3) * acc_noise_std
        return acc_noisy, done, info
